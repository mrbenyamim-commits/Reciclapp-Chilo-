# [ReciclApp Chiloé]

## 📱 Descripción del Proyecto
Esta aplicacion esta hecha con el fin de fomentar el reciclaje y dar a conocer los puntos de estos en el sector de gamboa alto, atravez de un mapa interactivo del sector

## 📂 Estructura del Proyecto
La arquitectura del código se encuentra organizada de manera lógica bajo el directorio `src/`, separando los componentes de la siguiente forma:

```text
├── assets/             # Imágenes, iconos y fuentes locales
├── src/
│   ├── components/     # Componentes funcionales reutilizables (Botones, Modales, etc.)
│   ├── screens/        # Pantallas principales (Login, Home, etc.)
│   └── navigation/     # Configuración de React Navigation (Rutas y flujos)
├── App.js              # Punto de entrada de la aplicación
├── package.json        # Dependencias del proyecto
└── README.md           # Documentación del proyecto
